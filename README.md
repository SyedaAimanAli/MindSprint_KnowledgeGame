MindSprint is a lightweight, gamified learning platform designed to make studying fast, fun, and motivating. Instead of long study sessions, users sprint through timed questions, earn XP, track accuracy, and improve with every run.

---

## Inspiration
Learning often feels boring and overwhelming. MindSprint was inspired by the idea that **short, competitive sprints** can keep users engaged and consistent.

---

## What it Does
- Runs quick quiz sessions with instant feedback  
- Tracks points, XP, accuracy, and correct answers  
- Uses gamification to keep users motivated and focused  

---

## How We Built It
MindSprint is built using **HTML, CSS, and JavaScript**, with a JSON-based question system and a clean, responsive UI optimized for both mobile and desktop.

---

## Challenges We Ran Into
- Balancing UI for both mobile and laptop views  
- Designing game logic that feels rewarding but simple  
- Keeping the experience fast without overwhelming users  

---

## Accomplishments We're Proud Of
- A fully working gamified quiz flow  
- Clean stats tracking (points, XP, accuracy)  
- A smooth, distraction-free learning experience  

---

## What We Learned
- Gamification can dramatically improve engagement  
- Small UI details have a big impact on usability  
- Simple ideas executed well can be powerful  

---

## What's Next for MindSprint
- Difficulty levels (Easy / Medium / Hard)  
- Daily challenges and streaks  
- User profiles and progress history  

---

## Getting Started
1. Clone the repository  
2. Open `main.html` in your browser  
3. Start sprinting
